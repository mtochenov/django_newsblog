Этот сайт-блог создан в учебных целях для зачета по предмету Разработка веб-приложений, группа ПРОГ-С-23, студент - Точенов Михаил.
На сайте реальзована модель CRUD, вы можете добавить новость, проситать ее, отредактировать или удалить.
Дополнительно на сайте реализовано меню для навигации.
Если новостей более 2-х (выбрал такое количество для размещения на 1-й странице), то для переключения добавляется пагинация.
Для того, чтобы сделать интерфейс более приятмы я подключил внешнюю JavaScript-библиотека Popper.js (версия 2.9.2) через CDN для управления всплывающими подсказками и динамическими элементами интерфейса.
